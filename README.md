## 一 项目说明
* 官网地址：https://reactnative.dev/docs/getting-started
* React Native中文网示例无法显示，故将官网示例操作演示
* 本实例包含：组件+APIS+Navigation

## 二 代码说明
* 将index.js中App替换为App1
* App1中将要显示的组件或API替换即可


## 三 使用说明
* 将项目copy下来
* 执行`yarn`安装依赖
* 将`export default StatusBarComponent`替换App.js中的组件

## 四 启动项目

```
npx expo start
```
